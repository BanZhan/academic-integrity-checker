# Sources Used For The Checker

Use these sources when updating the taxonomy or validation set:

- U.S. Office of Research Integrity, definition of research misconduct: https://ori.hhs.gov/definition-research-misconduct
- ORI administrative-action case summaries: https://ori.hhs.gov/content/case_summary
- ORI Brigidi case summary: https://ori.hhs.gov/content/case-summary-brigidi-gian-stefano
- ORI Zhang case summary: https://ori.hhs.gov/content/case-summary-zhang-liping
- ORI Chang case summary: https://ori.hhs.gov/content/case-summary-chang-alice-c
- ORI Sudbo case summary: https://ori.hhs.gov/case-summary-sudbo-jon
- ICMJE Recommendations, updated January 2026: https://www.icmje.org/recommendations/
- COPE Retraction Guidelines DOI: https://doi.org/10.24318/cope.2019.1.4
- Retraction Watch database and reporting on retractions: https://retractionwatch.com/
- Crossref REST API for bibliographic and DOI metadata checks: https://api.crossref.org/swagger-ui/index.html
- DOI.org resolution documentation and REST API: https://www.doi.org/the-identifier/resources/factsheets/doi-resolution-documentation
- OpenAlex API documentation for scholarly work metadata and abstracts: https://developers.openalex.org/
